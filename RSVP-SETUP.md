# Как подключить форму RSVP к Google-таблице

Ответы гостей (имя, «Буду / Не смогу», напитки, помощь с размещением, сообщение)
будут автоматически падать строками в твою Google-таблицу. Backend не нужен —
всё делает бесплатный Google Apps Script.

## Шаги (10 минут, один раз)

1. Создай новую Google-таблицу: https://sheets.new
   В первой строке (шапка) впиши по колонкам:
   `Дата | Имя | Статус | Напитки | Размещение | Сообщение | Страница`

2. В таблице открой меню **Расширения → Apps Script**.

3. Удали то, что там есть, и вставь код из файла [apps-script.gs](apps-script.gs)
   (он лежит рядом). Нажми **Сохранить** (значок дискеты).

4. Нажми **Развернуть → Новое развертывание**.
   - Тип: **Веб-приложение** (Web app).
   - «Выполнять от имени»: **Я** (Me).
   - «У кого есть доступ»: **Все** (Anyone).
   - Нажми **Развернуть**, дай разрешения (Google предупредит — это нормально,
     выбери свой аккаунт → «Дополнительно» → «Перейти … (небезопасно)» → Разрешить).

5. Скопируй **URL веб-приложения** — он заканчивается на `/exec`.

6. Открой `index.html`, найди строку:
   ```js
   var RSVP_ENDPOINT = "";
   ```
   и вставь URL в кавычки:
   ```js
   var RSVP_ENDPOINT = "https://script.google.com/macros/s/XXXXX/exec";
   ```

7. Закоммить и запушить (или попроси меня) — готово. Проверь: заполни форму на
   сайте → в таблице появится новая строка.

## Заметки
- Если поменяешь код скрипта — нужно снова **Развернуть → Управление развертываниями →
  Изменить → Новая версия**, иначе изменения не применятся.
- Форма отправляет данные в режиме `no-cors`, поэтому сайт не читает ответ сервера,
  но строка в таблицу записывается. Это нормально и ожидаемо.
- **Уведомления на почту** уже встроены: на каждый ответ придёт письмо. По
  умолчанию — на почту владельца таблицы. Хочешь другой адрес — впиши его в
  скрипте в строку `var NOTIFY_EMAIL = "";` и переразверни новую версию.
- Обновил код скрипта (например, добавил почту)? Обязательно переразверни новую
  версию — старый URL при этом не меняется, править сайт не нужно.
